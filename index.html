Pengenalan HTML dan CSS: Dasar-Dasar Web Development

1. Apa Itu HTML dan CSS? (Pengertian)

HTML (HyperText Markup Language) adalah bahasa markup standar yang digunakan untuk membuat struktur dasar halaman web. HTML berfungsi sebagai kerangka halaman, seperti judul, paragraf, gambar, dan tautan.

CSS (Cascading Style Sheets) adalah bahasa desain yang digunakan untuk mengatur tampilan dan gaya elemen HTML. Dengan CSS, kita bisa mengubah warna, ukuran, font, posisi, dan tata letak elemen halaman web.

2. Sejarah: Kenapa Ada HTML dan CSS?

  • HTML pertama kali dikembangkan oleh Tim Berners-Lee pada tahun 1991 untuk membuat dokumen hypertext di internet.
  • CSS diperkenalkan oleh Håkon Wium Lie pada tahun 1996 karena HTML terlalu terbatas dalam mengatur tampilan. Sebelumnya, semua gaya ditulis langsung di HTML.

Tanpa CSS, semua halaman web akan tampak sangat sederhana dan monoton.

3. Jenis-Jenis HTML dan CSS

Jenis HTML:

  • HTML4: Versi lama, digunakan tahun 1997–2014.
  • XHTML: Lebih ketat, mirip XML.
  • HTML5: Versi terbaru, mendukung video, audio, dan struktur semantik.

Jenis CSS:

  • Inline CSS: Ditulis langsung di dalam elemen HTML.
  • Internal CSS: Ditulis dalam tag <style> di dalam <head>.
  • External CSS: Ditulis dalam file .css terpisah dan dihubungkan dengan tag <link>.

4. Contoh Penulisan / Sintaks Utama

Contoh HTML:

<!DOCTYPE html>
<html>
<head>
  <title>Contoh HTML</title>
</head>
<body>
  <h1>Halo Dunia</h1>
  <p>Ini paragraf pertama.</p>
</body>
</html>

Contoh CSS:

body {
  background-color: #f0f0f0;
  font-family: Arial, sans-serif;
}

h1 {
  color: blue;
}

5. Cara Kerja + Diagram Masukan

  • HTML dan CSS dikirim dari server ke browser.
  • Browser membaca HTML untuk struktur, dan CSS untuk tampilan.
  • Kombinasi keduanya menghasilkan halaman yang utuh dan menarik.

8. Kelebihan dan Kekurangan

Kelebihan:

  • HTML: Mudah dipelajari, ringan, universal.
  • CSS: Memisahkan tampilan dari struktur, fleksibel, responsif.

Kekurangan:

  • HTML tidak bisa mengatur tampilan dengan baik tanpa CSS.
  • CSS bisa menjadi kompleks saat proyek besar.
  • Tidak interaktif (butuh JavaScript untuk fungsi dinamis).

9. Framework CSS yang Umum Digunakan

  • Bootstrap – framework populer untuk desain responsif.
  • Tailwind CSS – utility-first framework yang fleksibel.
  • Bulma – modern, ringan, dan mudah digunakan.
  • Foundation – mirip Bootstrap, tapi lebih kompleks.

Kesimpulan: HTML dan CSS adalah pondasi utama dalam membangun halaman web. Dengan HTML kita bisa menyusun konten, dan dengan CSS kita bisa mempercantik tampilan. Keduanya wajib dikuasai oleh setiap web developer pemula sebelum melangkah ke level lanjutan seperti JavaScript, framework, atau backend.
